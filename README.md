# Callum_96_97_Assessment
My year 12 Programming (AS 91896) and Processes (AS 91897) Assessment.

## Project Name
Workout Helper Program

## What the program does
This Python Tkinter program helps users track their workouts over time. Users can create an account, log in, continue as a guest, add workout entries, view workout history, filter workouts and track progress.

## Main features
- Login and create account system
- Guest mode
- Add workout screen
- Workout type and unit dropdowns
- Date input with automatic "/" formatting
- Input validation for blank fields, invalid dates and unrealistic amounts
- View workout history
- Filter workouts by exercise type
- Track progress and personal records
- Settings screen with colourblind mode and bigger text option
- Saves and loads data using text files

## Files used
- `workout_helper.py` - main Python program
- `Workouts.txt` - stores workout data
- `Users.txt` - stores user account details

## How to run the program
1. Open the project folder.
2. Open `workout_helper.py` in Python.
3. Run the file.
4. The program should open in a full screen Tkinter window.
